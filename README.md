# Drainfck

Brainf*ck interpreter in D. For getting hold in D.

## Building and Running

Drainfck supports two execution modes:

### 1️⃣ REPL Mode

```bash
dub
```

or 

```bash
dub build
./drainfck
```

Starts interactive Brainf*ck REPL.

### 2️⃣ Run a Source File

```bash
dub build
./drainfck run <source-file>
```

## Example

```bash
dub build
./drainfck run examples/hello_world.bf
```
